# V15 Florida Deals Network linking plan

No verified network domain inventory exists in this repository, so V15 does not publish speculative external destinations. This prevents broken links, accidental link-farm patterns, and unsupported ownership claims.

## Launch checklist for each destination

1. Confirm the property is live, useful, indexable, secure, and controlled by the intended publisher.
2. Record the canonical domain, destination purpose, responsible owner, and quarterly review date.
3. Add one contextual link from the most relevant OfferRadar section—not a sitewide list.
4. Attach `utm_source=offerradar`, `utm_medium=network`, and a descriptive campaign value if the destination analytics policy allows it.
5. Fire `network_outbound_click` with `network_destination`, `source_page`, and `link_position`.
6. Add a reciprocal link only where it gives readers a logical next step.
7. Run a scheduled HTTP/status/canonical audit and remove destinations that become stale.
